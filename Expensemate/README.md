# 💰 ExpenseMate

A beautiful web app to track your monthly income and expenses.

## 🌐 Live Demo
👉 [Try it here](https://three0-days-of-python-1khz.onrender.com)

## ✨ Features
- Set your monthly income
- Track fixed expenses (rent, subscriptions)
- Track general expenses (food, transport)
- See your remaining balance instantly
- Data saved between sessions

## 🛠️ Tech Stack
- Python
- Flask
- HTML & CSS
- Render (Deployment)

## 🚀 How to Run Locally
```bash
pip install -r requirements.txt
python app.py
```
